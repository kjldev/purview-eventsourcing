# Purview.EventSourcing.Shared

`Purview.EventSourcing.Shared` contains shared infrastructure and abstractions used by Purview EventSourcing provider packages.

## Install

```bash
dotnet add package Purview.EventSourcing.Shared
```

## What it provides

- Shared event store infrastructure used by provider implementations
- Common storage abstractions consumed across provider packages
- Reusable extension points that support provider-specific registrations
